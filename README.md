# Analizador Global de Indicadores de Desarrollo Sostenible (ONU)

## Descripción

Proyecto de Ciencia de Datos enfocado en analizar indicadores globales de desarrollo sostenible utilizando datos abiertos de organismos internacionales como la ONU y el Banco Mundial.

El backend se encarga del procesamiento, limpieza y análisis de datos utilizando Python.

## Objetivo General

Desarrollar un sistema capaz de procesar y analizar indicadores de desarrollo sostenible para facilitar la visualización y comparación entre países.

## Objetivos Específicos

- Obtener datos mediante APIs públicas.
- Procesar información utilizando Python.
- Analizar indicadores sociales y económicos.
- Preparar datos para visualizaciones.
